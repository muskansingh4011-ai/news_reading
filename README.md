# 📰 News Reader

A simple Python script that fetches the latest news articles using the [NewsAPI](https://newsapi.org).  
This project demonstrates how to build a basic news reader that constructs API URLs dynamically and retrieves results based on a search query.

---

## 🚀 Features
- Fetches news articles from NewsAPI.
- Allows you to specify a search query (e.g., "Samsung").
- Prints the generated API request URL for verification.
- Can be extended to parse JSON responses and display headlines.

---

## 📦 Requirements
- Python 3.7+
- [requests](https://pypi.org/project/requests/) library (for making API calls)
